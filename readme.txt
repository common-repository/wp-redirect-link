== Installation ==

The WP-Redirect-Link can uploaded plugin in Plugin Manager Page and Active.

1. Copy directory `WP-Redirect-Link` into your `wp-content/plugins` directory. You may wish to rename this to something else.
2. In the WordPress dashboard, navigation to the *Plugins* page
Locate the menu item that reads “The WordPress Plugin Boilerplate.”
3. Click on *Activate.*
